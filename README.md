# Readz

> **Un lector de EPUB pensado para desaparecer mientras leés.**

## Filosofía

Readz nace de una idea muy simple:

> **Nada debe interponerse entre el lector y el libro.**

No busca ser el lector con más funciones. Busca ser el lector más cómodo.

Cada decisión de diseño responde a una única pregunta:

> **¿Hace que leer durante horas sea una mejor experiencia?**

Si la respuesta es no, la función no forma parte de Readz.

---

# Objetivos

* Ser un lector de EPUB gratuito.
* Funcionar completamente en el navegador.
* Ser rápido, simple y elegante.
* Recordar la biblioteca y el progreso del lector.

---

# Filosofía del proyecto

Readz es una aplicación independiente.

Puede leer cualquier archivo EPUB.

```
Stories
    │
    ▼
 Readz
```

---

# Principios

* El libro es el protagonista.
* La interfaz debe desaparecer mientras se lee.
* Todo debe sentirse inmediato.
* Las configuraciones se guardan automáticamente.
* No existen anuncios.
* No existen funciones que distraigan.

---

# Experiencia de lectura

* Lectura por páginas (sin scroll).
* Navegación con teclado, mouse y gestos táctiles.
* Pantalla completa.
* Tema sepia por defecto.
* Modo claro y oscuro.
* Fuente Literata para el contenido.
* Interfaz minimalista.
* Controles visibles únicamente cuando el usuario interactúa.

---

# Biblioteca

Readz mantiene una biblioteca personal.

Los libros importados permanecen almacenados localmente para que puedan volver a abrirse sin tener que buscarlos nuevamente.

Cada libro conserva:

* progreso de lectura;
* última posición;
* marcadores;
* resaltados;
* notas;
* fecha de apertura.

---

# Nuestra biblioteca

Readz ofrecerá una colección opcional de libros de dominio público.

Los libros no vienen descargados.

El usuario decide cuáles incorporar a su biblioteca.

---

# Estadísticas

Readz guarda un historial de lectura.

Algunas estadísticas previstas:

* libros leídos;
* libros en lectura;
* páginas leídas;
* tiempo total de lectura;
* historial de libros terminados;
* fechas de inicio y finalización.

Las estadísticas forman parte del recorrido del lector, no de una competencia.

---

# Identidad

Readz transmite calma.

No habla demasiado.

No interrumpe.

Acompaña la lectura con pequeños mensajes cuando corresponde.

---

# Tecnologías

* HTML
* CSS
* JavaScript
* EPUB.js
* IndexedDB

Sin frameworks.

El objetivo es mantener un proyecto liviano, rápido y fácil de mantener.

---

# Visión

Readz no quiere competir por tener más funciones.

Quiere convertirse en ese lector que abrís, empezás a leer y, unos minutos después, olvidás que existe porque toda tu atención está puesta en el libro.
